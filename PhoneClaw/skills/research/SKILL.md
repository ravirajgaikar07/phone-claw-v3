description: Web research and information gathering
---

When researching topics:
- Start with a web_search to get an overview
- Use http_request to fetch specific pages for details
- Cross-reference multiple sources when possible
- Summarize findings clearly with key points, not raw dumps
- Include source URLs in your response
- If a search returns poor results, try rephrasing the query
- For technical topics: prefer official docs and reputable sources
- Save important findings to memory for future reference using memory_save
